> Installation
```cargo install perorustbot```

> list of commands
+ say
+ help (list of all commands)
+ ping
+ ipi (soon)
+ follow (soon)
+ cs (corona stats)
+ ani (anime info)
+ ctid (chat id, your id, replied users id)
+ udi (urban dict)
